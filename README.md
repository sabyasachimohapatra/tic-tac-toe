## Description

This project is a web-based implementation of the classic Tic Tac Toe game. Players take turns to mark the spaces in a 3Ã—3 grid with 'X' and 'O'. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features
```
- Two-player mode: Play against another person.
- Interactive game grid with clickable cells.
- Displays current player information.
- Highlights the winning combination.
- Displays a message for a tie game.
- Restart button to start a new game.
```
